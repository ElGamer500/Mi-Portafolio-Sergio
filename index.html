Check intime

Check Intime es una herramienta de control de asistencia para tiendas. Permite a los empleados registrar sus horas de entrada y salida, ver su historial de asistencia, ausencias y descuentos. Los administradores pueden monitorear la actividad de todos los empleados, mientras que los usuarios solo pueden ver, pero no modificar, su propia actividad.

Tecnologías Utilizadas

Visual Studio con Windows Forms, C#, SQL Server

Mundo Autopartes

Mundo Autopartes es una página web para la venta de autopartes eléctricas. La plataforma permite a los usuarios registrarse, iniciar sesión y comprar partes eléctricas. Los administradores pueden gestionar productos y proveedores, facilitando la administración y actualización de la información de manera sencilla y eficiente.

Tecnologías Utilizadas

HTML5, CSS, Bootstrap, Node.js, EJS, JavaScript, MySQL
